ボイスチャンネルへの入退室を記録します．

== 操作方法

入退室のログを取りたいメンバーに，ロール `入退室ログ` を付与してください．

== 必要な権限

Manage Roles:: ログを残すアカウントを管理するために，権限を作成します．
Manage Channels:: ログを残すチャンネルを作成します．
Send Messages:: ログを送信します．
